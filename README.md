**Пространственная репрезентация российских городов в песенных текстах: анализ топонимов с использованием LLM**

Работа посвящена разработке и оценке подхода к извлечению топонимов из русскоязычных песенных текстов, где пространственные упоминания часто даны не в официальной, а в разговорной, сленговой или контекстной форме. Основная задача исследования — адаптировать LLM под этот нестандартный корпус, чтобы автоматически выделять, нормализовать и затем геокодировать такие упоминания для последующего пространственного и содержательного анализа образов города в песнях.
В качестве основного подхода используется дообучение open-source LLM на размеченном корпусе песенных текстов с сопоставлением результатов базовой и адаптированной модели. Методически работа строится как пайплайн из распознавания топонимов, их нормализации, геокодирования и последующего анализа с оценкой качества по метрикам precision, recall и F1-score.
